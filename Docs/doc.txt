==========Campos==========
id = identificador
titulo 
descricao 
videoId = trailer do jogo


=========Validacao===============
id = numero > 0
titulo = > 3 && <= 100
descricao = >= 10 && <= 250
videoId = != "" && <= 15


=========ENDPOINT========
GET    = /game     | retorna TODOS
GET    = /game/:id | retorna game pelo id
POST   = /game     | Criar um novo game
PUT    = /game/:id | Altera um game
DELETE = /game/:id | deleta um game
